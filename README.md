## 情報の例

- 太郎（Taro, 1950年生まれ）の子供は、一郎（Ichiro, 1975年生まれ）と次郎（Jiro, 1977年生まれ）である。
- 一郎の子供は、三郎（Saburo, 2000年生まれ）である。
- 次郎の子供は、四郎（Shiro, 2002年生まれ）である。

※各人物は自身の子供のリストを保持する。太郎は一郎と次郎のリストを持ち、一郎は三郎のリストを持ち、次郎は四郎のリストを持つ。三郎と四郎は空のリストを持つ。

## 計算の例

太郎を起点とする家系図において、指定した2人の人物がいとこ関係にあるかを判定する。

- 三郎と四郎について判定した場合、それぞれの親である一郎と次郎が兄弟関係にあるため、「いとこである」という結果（真）が得られる。
- 一郎と次郎について判定した場合、この2人は兄弟であっていとこではないため、「いとこではない」という結果（偽）が得られる。
- 一郎と三郎について判定した場合、親と子の関係であるため、「いとこではない」という結果（偽）が得られる。
